## Быстрый старт

```bash
# Установка зависимостей
pnpm install

# Запуск в режиме разработки (порт 8080)
pnpm dev

# Сборка для продакшена
pnpm build

# Запуск продакшен-сервера
pnpm start
```

Приложение будет доступно по адресу `http://localhost:8080`.

Для входа в систему используйте демо-аккаунт:
- **Email:** `demo@example.com`
- **Пароль:** `123456`

---

## Что умеет приложение

- Просмотр галереи 3D-моделей с превью
- Интерактивный просмотрщик 3D (вращение, зум, панорамирование мышью)
- Регистрация и вход пользователей
- Загрузка новых моделей (через профиль)
- Просмотр детальной страницы каждой модели
