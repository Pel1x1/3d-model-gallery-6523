# Визуализатор 3D-моделей в браузере

Веб-приложение для загрузки, хранения и визуализации 3D-моделей в браузере.

## Технологии
- React — клиентская часть
- Node.js — серверная часть
- Node.js + SQL — работа с базой данных

## Структура проекта
- client/ — клиентская часть
- server/ — серверная часть
- database/ — работа с базой данных
